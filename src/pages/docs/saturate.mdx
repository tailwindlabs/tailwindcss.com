## Usage

Use the `saturate-{amount}` utilities alongside the `filter` utility to control an element's saturation.

```html rose
<template preview>
  <div class="grid grid-cols-2 gap-2 sm:grid-cols-5">
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-rose-400 rounded-md filter saturate-0"></div>
      <p class="relative font-extrabold text-white">0</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-rose-400 rounded-md filter saturate-50"></div>
      <p class="relative font-extrabold text-white">50</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-rose-400 rounded-md filter saturate-100"></div>
      <p class="relative font-extrabold text-white">100</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-rose-400 rounded-md filter saturate-150"></div>
      <p class="relative font-extrabold text-white">150</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-rose-400 rounded-md filter saturate-200"></div>
      <p class="relative font-extrabold text-white">200</p>
    </div>
  </div>
</template>

<div class="filter **saturate-0** ..."></div>
<div class="filter **saturate-50** ..."></div>
<div class="filter **saturate-100** ..."></div>
<div class="filter **saturate-150** ..."></div>
<div class="filter **saturate-200** ..."></div>
```

## Responsive

To control an element's saturation at a specific breakpoint, add a `{screen}:` prefix to any existing saturate utility. For example, use `md:saturate-150` to apply the `saturate-150` utility at only medium screen sizes and above.

```html
<div class="filter saturate-110 **md:saturate-150** ...">
  <!-- ... -->
</div>
```

For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation.
