## Examples

### Basic example

Use `min-inline-<number>` utilities like `min-inline-24` and `min-inline-64` to set an element to a fixed minimum inline size based on the spacing scale:

<Figure>

<Example>
  {
    <div className="relative mx-auto grid w-60 justify-items-start gap-y-4 text-center font-mono text-xs font-bold text-white">
      <Stripes border className="absolute -inset-y-4 w-20 rounded-lg" />
      <div className="relative min-inline-80 rounded-lg bg-sky-500 px-4 py-2">min-inline-80</div>
      <div className="relative min-inline-64 rounded-lg bg-sky-500 px-4 py-2">min-inline-64</div>
      <div className="relative min-inline-48 rounded-lg bg-sky-500 px-4 py-2">min-inline-48</div>
      <div className="relative min-inline-40 rounded-lg bg-sky-500 px-4 py-2">min-inline-40</div>
      <div className="relative min-inline-32 rounded-lg bg-sky-500 px-4 py-2">min-inline-32</div>
      <div className="relative min-inline-24 rounded-lg bg-sky-500 px-4 py-2">min-inline-24</div>
    </div>
  }
</Example>

### Using a percentage

Use `min-inline-full` or `min-inline-<fraction>` utilities like `min-inline-1/2` and `min-inline-2/5` to give an element a percentage-based minimum inline size:

<Figure>

<Example>
  {
    <div className="relative mx-auto my-6 flex w-full justify-items-start gap-3 text-center font-mono text-xs font-bold text-white">
      <Stripes border className="absolute inset-x-0 -inset-y-3 rounded-lg" />
      <div className="relative min-inline-2/3 rounded-lg bg-indigo-500 px-4 py-2">min-inline-3/4</div>
      <div className="relative inline-full rounded-lg bg-indigo-300 px-4 py-2 dark:bg-indigo-800 dark:text-indigo-400">
        inline-full
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:min-inline-3/4] -->

  <div class="min-inline-3/4 ...">min-inline-3/4</div>
  <div class="inline-full ...">inline-full</div>
</div>
```

</Figure>
