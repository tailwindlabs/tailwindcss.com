## Examples

### Basic example

Use `min-block-<number>` utilities like `min-block-24` and `min-block-64` to set an element to a fixed minimum block size based on the spacing scale:

<Figure>

<Example>
  {
    <div className="mx-auto flex justify-center">
      <div className="relative flex items-end space-x-4 font-mono text-xs font-bold text-white">
        <Stripes border className="absolute -inset-x-[5%] block-20 w-[110%] rounded-lg" />
        <div className="relative flex min-block-96 w-8 items-end justify-center rounded-lg bg-blue-500 pb-8 [writing-mode:sideways-lr]">min-block-96</div>
        <div className="relative flex min-block-80 w-8 items-end justify-center rounded-lg bg-blue-500 pb-8 [writing-mode:sideways-lr]">min-block-80</div>
        <div className="relative flex min-block-64 w-8 items-end justify-center rounded-lg bg-blue-500 pb-8 [writing-mode:sideways-lr]">min-block-64</div>
        <div className="relative flex min-block-48 w-8 items-end justify-center rounded-lg bg-blue-500 pb-8 [writing-mode:sideways-lr]">min-block-48</div>
        <div className="relative flex min-block-40 w-8 items-end justify-center rounded-lg bg-blue-500 pb-8 [writing-mode:sideways-lr]">min-block-40</div>
        <div className="relative flex min-block-32 w-8 items-end justify-center rounded-lg bg-blue-500 pb-8 [writing-mode:sideways-lr]">min-block-32</div>
        <div className="relative flex min-block-24 w-8 items-end justify-center rounded-lg bg-blue-500 pb-8 [writing-mode:sideways-lr]">min-block-24</div>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:min-block-80,min-block-64,min-block-48,min-block-40,min-block-32,min-block-24] -->
```

### Using a percentage

Use `min-block-full` or `min-block-<fraction>` utilities like `min-block-1/2`, and `min-block-2/5` to give an element a percentage-based minimum block size:

<Example>
  {
    <div className="flex block-96 items-end justify-center space-x-4 font-mono text-xs font-bold text-white">
      <div className="relative flex block-full items-end">

        <div className="relative flex min-block-full w-8 items-end justify-center rounded-lg bg-sky-500 pb-10 [writing-mode:sideways-lr]">min-block-full</div>
      </div>
      <div className="relative flex block-full items-end">
        <Stripes border className="absolute inset-0 block-full rounded-lg" />

      </div>
      <div className="relative flex block-full items-end">
        <Stripes border className="absolute inset-0 block-full rounded-lg" />
        <div className="relative flex min-block-3/4 w-8 items-end justify-center rounded-lg bg-sky-500 pb-10 [writing-mode:sideways-lr]">min-block-3/4</div>
      </div>
      <div className="relative flex block-full items-end">
        <Stripes border className="absolute inset-0 block-full rounded-lg" />
