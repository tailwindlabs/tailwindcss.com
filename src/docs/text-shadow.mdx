## Examples

### Basic example

Use utilities like `text-shadow-sm` and `shadow-lg` to apply different sized text shadows to a text element:

<Figure>

<Example padding={false}>
  {
    <div className="grid items-center justify-around gap-4 bg-gray-100 px-6 py-14 text-center font-bold text-indigo-600">
      <p className="text-xl text-shadow-2xs">The quick brown fox jumps over the lazy dog.</p>
      <p className="text-2xl text-shadow-xs">The quick brown fox jumps over the lazy dog.</p>
      <p className="text-3xl text-shadow-sm">The quick brown fox jumps over the lazy dog.</p>
      <p className="text-4xl text-shadow-md">The quick brown fox jumps over the lazy dog.</p>
      <p className="text-5xl text-shadow-lg">The quick brown fox jumps over the lazy dog.</p>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-shadow-md,text-shadow-lg,text-shadow-xl] -->
<div class="text-shadow-md ..."></div>
<div class="text-shadow-lg ..."></div>
<div class="text-shadow-xl ..."></div>
```

</Figure>

### Setting the shadow color

Use utilities like `text-shadow-indigo-500` and `text-shadow-cyan-500/50` to change the color of a text shadow:

<Figure>

<Example>{}</Example>

```html
<!-- [!code classes:text-shadow-cyan-500/50,text-shadow-blue-500/50,text-shadow-indigo-500/50] -->
```

</Figure>

By default colored shadows have an opacity of 100% but you can adjust this using the opacity modifier.
