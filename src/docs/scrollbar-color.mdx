### Using a custom value

Use utilities like `scrollbar-thumb-[<value>]` and `scrollbar-track-[<value>]` to set scrollbar colors based on
completely custom values:

```html
<!-- [!code classes:scrollbar-thumb-[#0f766e],scrollbar-track-[#ccfbf1]] -->
<div class="scrollbar-thumb-[#0f766e] scrollbar-track-[#ccfbf1] ...">
  <!-- ... -->
</div>
```

For CSS variables, you can also use the `scrollbar-thumb-(<custom-property>)` and
`scrollbar-track-(<custom-property>)` syntax:

```html
<!-- [!code classes:scrollbar-thumb-(--scrollbar-thumb),scrollbar-track-(--scrollbar-track)] -->
<div class="scrollbar-thumb-(--scrollbar-thumb) scrollbar-track-(--scrollbar-track) ...">
  <!-- ... -->
</div>
```

## Customizing your theme

Use the `--scrollbar-thumb-color-*` and `--scrollbar-track-color-*` theme variables to customize the scrollbar color
utilities in your project:

<div>
  <div className="not-prose">
    <CodeExample
      example={{
        lang: "css",
        code: `@theme {
  --scrollbar-thumb-color-brand: #0f766e; /* [!code highlight] */
  --scrollbar-track-color-brand: #ccfbf1; /* [!code highlight] */
}`,
      }}
    />
  </div>
</div>

Now utilities like `scrollbar-thumb-brand` and `scrollbar-track-brand` can be used in your markup:

```html
<!-- [!code classes:scrollbar-thumb-brand,scrollbar-track-brand] -->
<div class="scrollbar-thumb-brand scrollbar-track-brand ...">
  <!-- ... -->
</div>
```

These utilities also support your existing color palette using the `--color-*` theme variables.
